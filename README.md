## 🚀 Getting Started

To set up the project locally:

### Prerequisites

- Node.js (v14 or above)
- npm or yarn

### Installation

```bash
git clone https://github.com/your-username/your-portfolio-repo.git
cd your-portfolio-repo
npm install
```

### Running the App

```bash
npm run dev
```

Visit `http://localhost:3000` to view the application in the browser.
